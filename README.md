# LLM-Web-Chatbot 🤖
An AI assistant chatbot using Google Gemini, Streamlit, conversation memory, and web search integration.

## 📁 Project Overview

This chatbot provides:

✅ AI Question Answering  
✅ Conversation Memory / Context Handling  
✅ Chat History Management  
✅ Web Search Assisted Responses  
✅ Attractive HTML + CSS Interface  
✅ Streamlit Interactive UI
