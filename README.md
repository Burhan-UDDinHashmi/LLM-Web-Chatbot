# LLM-Web-Chatbot
An AI assistant chatbot using Google Gemini, Streamlit, conversation memory, and web search integration.

This chatbot can:
- Answer user questions
- Maintain conversation history
- Fetch Google search results
- Generate AI-powered responses

## Features

- Modern chatbot UI
- Conversation memory
- Google search integration
- Gemini AI responses
- Streamlit-based frontend

## Technologies Used

- Python
- Streamlit
- Google Gemini API
- Google Search
